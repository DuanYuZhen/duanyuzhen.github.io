产品

发表于 2017-04-17   |   分类于 产品   |     |   次阅读   |   阅读次数

帮用户尽快作出购买决策:
1.广告推荐——>进入网站

2.挑选商品：
（1）购买目标明确：直接搜索——>搜索品牌、搜索品类
（2）购买目标不明确：先看导购——>主要看的是首页内容，营销导购、内容导购
导购商品的目的：让用户快速购买商品 导购商品的手段：折扣促销、特价、抢购等
*导购商品的关键：首屏的钩子模块
（a）产品中常用的钩子模块有：秒杀模块、今日爆款、人气冠军、抢红包、领金币、猜你喜欢、限时抢、整点抢购、必抢等。
（b）钩子模块作用：吸引用户点击、降低跳失率、强化用户对平台的认知。

3.决定购买（价格因素、信任因素(买过用户的评价)、卖家的承诺）

4.再次购买：
进行使用红包、优惠券或者积分
还有就是老用户的定期唤醒（通过推送活动、优惠等）

5.帮用户找到购买理由：
（a）价格刺激——省钱
（b）优惠刺激——省钱
（c）库存数量刺激——量少
（d）打消用户的顾虑——买过的用户的评论
